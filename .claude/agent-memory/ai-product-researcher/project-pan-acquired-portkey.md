**Palo Alto Networks acquired Portkey** — announced 2026-04-30, closed 2026-05-29; Portkey valued ~$120–140M. Portkey is now the AI Gateway for Prisma AIRS ("central nervous system" for securing AI agents). Portkey = the open-source gateway Routeplane explicitly clones per ADR-009.

**Why:** This is the single most important market event for Routeplane's thesis. It simultaneously (a) *validates* "agentic security gateway" as the real category and (b) installs a $100B+ incumbent on the exact hill Routeplane wants. The feature-matrix claim "category is open / (none)" is now only half true — open as a label, closing as a defensible position.

**How to apply:** Any future positioning/strategy call must treat PAN/Portkey as the primary competitor, not 2024-era "scrappy Portkey." The watch-trigger that escalates to a moat crisis: PAN/Portkey shipping India-resident hosting + per-request residency classification. Counter-positioning (Helmer) is the defence — PAN is a US SaaS control plane that India BFSI compliance will distrust; see [[project-market-assessment-positioning]].
